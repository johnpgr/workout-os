# Workout OS

App de treino utilizando React com Tailwind e componentes shadcn/ui, mantendo:

- plano Push / Pull / Legs
- calendário semanal (modo 6x e 3x)
- logs por sessão com `IndexedDB`
- resumo semanal e histórico por dia

## Rodando localmente

```bash
pnpm install
pnpm dev
```

## Build estático

```bash
pnpm build
```

A saída estática é gerada em `dist/`.
